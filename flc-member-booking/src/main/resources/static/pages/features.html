Application Features & Help
Overview

This application provides a simple booking system for FLC members. Use the navigation to view members and lessons, book lessons, manage bookings, and run monthly reports.

Main Features
  • Members: View and search members. The members table is used to populate the booking form.
  • Lessons: Browse all lessons in a table. Use filters (day, exercise) and quick search to narrow results. Click Select on a lesson to pre-fill the booking form.
  • Book: Choose a member and lesson ID (auto-filled from Lessons) to create a booking.
  • Manage Booking: Change, cancel, or mark attendance (with rating & review) for bookings.
  • Reports: Generate monthly lesson and income reports.
Notes & Tips
  • The header and footer are fixed; the main content area between them scrolls independently.
  • Filters on the Lessons panel include client-side Apply (combines day/exercise/free-text) and a Refresh button that fetches the latest data from the server.
  • If a lesson's seats are 0, the system will still allow selecting the lesson but booking may fail depending on server-side validation.
  • For support, contact support@flc.example.
Endpoints used

(For reference) The UI uses these API endpoints: /api/members, /api/lessons, /api/bookings, and reporting endpoints under /api/reports.
